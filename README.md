# Personal Portfolio Website

## Overview

This project is my personal portfolio website built to showcase my skills, projects, and professional background. The website highlights my journey as a **Biological Science student and aspiring Full-Stack Web Developer** while demonstrating my frontend development abilities.

The portfolio presents information about me, my technical skills, development experience, and provides ways for visitors or potential employers to contact me.

---

## Features

* Responsive design that works on **desktop, tablet, and mobile devices**
* **About Me section** introducing my background and goals
* **Skills section** with progress bars for frontend and backend technologies
* **Experience section** describing my Full-Stack Web Development training
* **Certificate display** for my Udemy Full-Stack Web Developer certificate
* **Contact form** for visitors to send messages
* **Get in Touch section** with my email and phone number
* **Smooth scrolling navigation**
* **Back to Top button**
* Clean and modern UI layout
